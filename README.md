# CAPSTONE

## Rent Payment App

This project is a web-based rent payment recording system developed as part of a capstone requirement.

The application allows tenants to submit rent payment details through a simple and accessible online form. Submitted data is automatically recorded in a centralized Google Sheet, providing the property owner with a structured, real-time record of payments.

### Key Features
- Tenant-friendly rent payment submission form
- Automatic logging of payment records to Google Sheets
- Support for multiple payment methods (cash, e-wallet, bank transfer)
- Optional upload of proof of payment
- Timestamped records for tracking and verification

### Objective
The goal of this project is to reduce manual rent collection, minimize recording errors, and improve transaction efficiency for small property owners by using a low-cost, digital solution.

### Technologies Used
- Python
- Streamlit
- Google Sheets API
- Google Drive API (optional for proof uploads)

This system is designed to be easy to use, scalable, and suitable for non-technical users.
